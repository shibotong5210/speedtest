就是需要一个追尾预警系统，有激光测距传感器，车速传感器，油门踏板传感器（压力或者角度的就行我感觉）
，oled屏，刚开始测车速，车速小于一定值不报警，车速大于一定值的时候去测车距，车距小于一定值的时候报警，
如果这时候监测到油门松开了就停止报警，油门没松开就持续报警
